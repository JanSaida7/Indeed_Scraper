# Indeed Scraper

A web application to scrape and view job listings from Indeed, using Apify and PostgreSQL.

## Setup

1.  **Install Dependencies:**
    ```bash
    pip install -r requirements.txt
    ```

2.  **Configure Environment:**
    Ensure you have a `.env` file with your Apify token and Database credentials.

3.  **Run the Database:**
    Make sure your PostgreSQL database is running.

## Usage

1.  **Start the Application:**
    ```bash
    python app.py
    ```

2.  **Access the Dashboard:**
    Open your browser and navigate to `http://127.0.0.1:5001`.
